This directory is a copied documentation snapshot from the upstream
`gurdasnijor/durable-streams` repository. It is not the canonical protocol
source for this repository.

Canonical local sources remain:

- `PROTOCOL.md`
- `typespec/`
- `docs/sdds/`
- package source and tests under `packages/`

The snapshot preserves upstream documentation context for server/client design,
conformance history, package READMEs, examples, skills, and protocol-adjacent
notes. When upstream reference material conflicts with local protocol or
TypeSpec files, the local files win.

The upstream VitePress executable entrypoints under `docs/.vitepress` are not
copied because this repository treats the snapshot as reference documentation,
not as an active website build.
